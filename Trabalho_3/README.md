# Trabalho III - Backend Base

Projeto-esqueleto para o Trabalho III.

O objetivo desta pasta e oferecer:

- uma estrutura inicial alinhada com as Aulas 10, 11 e 12;
- contratos e pontos de entrada principais;
- testes unitarios e de integracao que funcionam como especificacao executavel.

## Importante

- os arquivos de dominio, casos de uso e repositorios foram deixados sem implementacao de negocio;
- a suite de testes foi criada para falhar inicialmente;
- a meta do grupo e implementar o backend ate fazer os testes passarem.

## Comandos

```bash
npm install
npm test
npm run dev
```
